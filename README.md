## Dev Dependencies
[NodeJS](https://nodejs.org/en/) - LTS Version<br />
[NestJS CLI](https://docs.nestjs.com/cli/overview)<br />
[Visual Studio Code](https://code.visualstudio.com/) or another editor as your preference

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start
# watch mode
$ npm run start:dev
# production mode
$ npm run start:prod
```
